# Netflix Movie Data Analysis

This project analyzes movie data using Python and data visualization libraries. The dataset was cleaned, explored, and visualized to identify trends in movie popularity, ratings, genres, and audience preferences.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Highlights
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Movie Popularity Analysis
- Genre-Based Insights
- Data Visualization

This project demonstrates practical data analysis and visualization skills using real-world movie data.
